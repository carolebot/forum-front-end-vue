# Restaurant forum
- A Practice for Vue

## Features
- Browse restaurants & show details
- Add & remove Favorite restaurants
- Like or Unlike restaurants
- Follow & Unfollow users
- Add & Delete comments
- Edit personal profile

## How to open
- git clone
- npm install
- npm run serve
- open http://localhost:8080 

## What I used
- @vue/cli 4.5.15, with BootStrap
